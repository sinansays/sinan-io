Colophon

This site is hand-built with plain HTML and CSS, with just enough vanilla JS to support a global header and footer. I have intentionally avoided using frameworks, build tooling, or heavy client-side dependencies. My design decisions centered on performance, longevity, accessibility, and agentic composability.

A CSS-only theme toggle flips color variables, while Cloudflare Pages serves the repository exactly as written for fast delivery.

Typography leans on a system-first sans-serif stack—San Francisco on Apple hardware and Inter elsewhere—paired with a refined SF-inspired monospace for code and technical accents.
